# Milestone Project: Flashcard Trainer

Build a study app for question-answer cards.

## Required Features

- Create decks.
- Add cards.
- Quiz the user.
- Shuffle questions.
- Track correct and incorrect answers.
- Write tests for scoring and deck loading.

## Stretch Goals

- Add spaced repetition buckets.
- Import cards from CSV.
- Show stats by deck.

## Completion Criteria

- At least 8 unit tests exist.
- Repeated quiz sessions produce sensible stats.
- The code uses comprehensions or iterator tools where they improve clarity.
