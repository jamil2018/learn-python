# Module Index

Each module folder contains:

- Topic folders such as `01-string-basics/`.
- One subfolder per problem.
- A `problem.md` file inside each problem folder.
- `notes.md`: core concepts for that module.
- `99-milestone-project/README.md`: the project to build after finishing the module.

Work in order. The modules intentionally repeat small ideas in new contexts so the syntax becomes muscle memory.
