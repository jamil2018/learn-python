# Milestone Project: Command-Line Toolkit Package

Turn your earlier utilities into a package with subcommands.

## Required Features

- Package can be run with `python -m toolkit`.
- Subcommands for `convert`, `text`, and `stats`.
- Shared formatting helpers live in a separate module.
- CLI code is separate from pure calculation functions.
- At least one command supports a `--json` output flag.

## Stretch Goals

- Add a config file.
- Add a `--verbose` mode.
- Add a simple table printer.

## Completion Criteria

- The package has more than one module.
- Commands show useful help text.
- Pure functions can be imported and reused elsewhere.
