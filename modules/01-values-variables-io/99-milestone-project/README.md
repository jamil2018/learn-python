# Milestone Project: Unit Converter

Build a terminal program that converts everyday values.

## Required Features

- Convert Celsius to Fahrenheit.
- Convert Fahrenheit to Celsius.
- Convert minutes into hours and minutes.
- Convert seconds into days, hours, minutes, and seconds.
- Calculate a tip and final bill total.
- Let the user choose an operation from a menu.

## Constraints

- Use only concepts from Module 1.
- Keep everything in one Python file for now.
- Invalid numeric input may crash; you will learn better validation later.

## Completion Criteria

- The program runs more than one conversion without restarting.
- Output is formatted clearly.
- Each menu option works with at least 2 manual test examples.
