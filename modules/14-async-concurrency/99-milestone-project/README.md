# Milestone Project: Async Website Checker

Build a tool that checks a list of URLs concurrently.

## Required Features

- Read URLs from a text file.
- Check status codes concurrently.
- Show response time.
- Mark failed requests clearly.
- Sort results by response time.

## Stretch Goals

- Add timeout configuration.
- Save results as CSV.
- Add retry logic.

## Completion Criteria

- A slow URL does not block every other URL.
- Failures are captured as results.
- The concurrency limit can be changed.
