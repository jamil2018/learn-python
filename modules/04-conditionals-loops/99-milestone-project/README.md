# Milestone Project: Number And Word Games

Build a menu-driven game collection.

## Required Features

- Number guessing game.
- Rock-paper-scissors with multiple rounds.
- Palindrome challenge.
- FizzBuzz printer with a user-selected limit.
- Score tracking across games.

## Constraints

- Use conditionals and loops heavily.
- Store scores in dictionaries or lists.
- Keep it terminal-based.

## Completion Criteria

- Each game can be played repeatedly.
- Invalid choices are handled with another prompt.
- The user can quit from the main menu.
