# Milestone Project: Personal CLI Toolkit

Build a single Python script with small utilities you can run from a menu.

## Required Features

- Convert temperatures between Celsius, Fahrenheit, and Kelvin.
- Convert text to title case, snake case, and kebab case.
- Count words, characters, and unique words in pasted text.
- Maintain an in-memory list of favorite commands.
- Let the user repeatedly choose actions until they quit.

## Stretch Goals

- Add a search command for favorite commands.
- Add a command history list.
- Add input validation for empty text and invalid numbers.

## Completion Criteria

- The program has a clear menu.
- Related values are stored in lists or dictionaries.
- Invalid menu choices do not crash the program.
