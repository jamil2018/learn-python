# Milestone Project: Log Analyzer

Build a tool that summarizes log-like text files.

## Required Features

- Read a log file line by line.
- Count log levels such as `INFO`, `WARN`, and `ERROR`.
- Find the top repeated messages.
- Filter lines by keyword.
- Print a sorted summary.

## Stretch Goals

- Support multiple input files.
- Add a `--level` CLI filter.
- Process large files without loading everything into memory.

## Completion Criteria

- The program uses Python iteration tools where they improve clarity.
- Large input files are handled sensibly.
- Output is readable and consistently ordered.
