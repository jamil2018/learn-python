# Milestone Project: CSV Report Builder

Build a tool that cleans a CSV file and creates reports.

## Required Features

- Read a messy CSV file.
- Clean missing values and string numbers.
- Normalize category names.
- Group records by month and category.
- Generate a top-10 report.
- Export cleaned CSV and summary JSON.

## Stretch Goals

- Add duplicate detection.
- Add moving averages.
- Add a `--sort` option for reports.

## Completion Criteria

- Data cleaning logic is tested.
- The original file is not overwritten.
- The final report answers useful questions about the data.
