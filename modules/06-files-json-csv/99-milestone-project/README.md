# Milestone Project: File-Backed Todo List

Build a todo list that survives after the program closes.

## Required Features

- Add todo items.
- List all todo items.
- Mark an item complete.
- Delete an item.
- Save todos to JSON.
- Load todos when the program starts.

## Stretch Goals

- Add due dates.
- Export todos to CSV.
- Search todos by keyword.

## Completion Criteria

- Restarting the program does not lose todos.
- Data is stored in a readable JSON file.
- File reading and writing are isolated in small functions.
