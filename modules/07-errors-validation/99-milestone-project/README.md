# Milestone Project: Expense Ledger

Build a file-backed expense tracker.

## Required Features

- Add an expense with date, category, description, and amount.
- Save expenses to JSON or CSV.
- Load existing expenses when the program starts.
- Show totals by category.
- Show monthly total.
- Reject invalid dates and invalid amounts.

## Stretch Goals

- Export a summary report.
- Add edit and delete commands.
- Add budget warnings per category.

## Completion Criteria

- Restarting the program does not lose data.
- Corrupt or missing data files are handled gracefully.
- Money is formatted consistently.
