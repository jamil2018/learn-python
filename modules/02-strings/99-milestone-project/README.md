# Milestone Project: Text Formatter

Build a terminal tool that accepts text and transforms it.

## Required Features

- Show character count, word count, and unique word count.
- Convert text to uppercase, lowercase, title case, snake case, and kebab case.
- Mask a sensitive number, preserving only the final 4 characters.
- Check whether a word or phrase is a palindrome.
- Generate initials from a full name.

## Constraints

- Use strings, numbers, variables, and basic loops only if needed.
- Keep each transformation easy to run from a menu.

## Completion Criteria

- The program can process at least 5 different text operations.
- Empty strings are handled with a clear message.
- You include sample inputs and outputs in comments or a small `README.md`.
