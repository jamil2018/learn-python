# Milestone Project: Personal Knowledge Index

Build a local search tool for Markdown notes.

## Required Features

- Scan a folder of `.md` files.
- Extract headings and links.
- Search by keyword.
- Show matching file, line number, and preview.
- Rank exact heading matches above body matches.

## Stretch Goals

- Add tags from frontmatter.
- Add fuzzy matching.
- Add a simple interactive prompt.
- Write an index file to speed up future searches.

## Completion Criteria

- The design uses small functions or classes with clear responsibilities.
- The search result format is consistent.
- The project feels like something you would actually reuse.
