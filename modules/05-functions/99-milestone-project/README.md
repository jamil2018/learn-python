# Milestone Project: Text Adventure Engine

Build a small terminal adventure game with rooms, items, and branching choices.

## Required Features

- At least 6 rooms.
- Player inventory.
- Locked path that requires an item.
- Win and lose states.
- Help command that lists available actions.
- Functions for command parsing, moving, taking items, and displaying state.

## Stretch Goals

- Store room data in dictionaries instead of hardcoded `if` chains.
- Add health or score.
- Add commands like `take key` and `go north`.

## Completion Criteria

- Game state is stored clearly.
- Repeated commands work.
- Player cannot enter impossible states.
- No single function does everything.
