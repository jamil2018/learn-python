# Milestone Project: Contact Book Package

Build a small package, not just one script.

## Required Features

- Add, list, search, update, and delete contacts.
- Store contacts as dataclasses.
- Split code into modules such as `models.py`, `storage.py`, `cli.py`.
- Save data to JSON.
- Provide a command-line interface.

## Stretch Goals

- Add tags.
- Import contacts from CSV.
- Add birthday or notes fields.

## Completion Criteria

- The app can be run with `python -m contact_book`.
- Data logic and CLI printing are not tangled together.
- Contact search is case-insensitive.
